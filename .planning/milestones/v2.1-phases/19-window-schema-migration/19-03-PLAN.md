## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| non-staff request → per-campaign table SELECT | The queryset must never fetch contact PII for non-staff |
| public submitter → CampaignRunSubmissionForm | Untrusted free-text/date input crosses into CampaignRun.create |
| approve click → sun_event()/Observatory data | Messy Observatory records (blank timezone) can raise mid-projection |
