<objective>
Create the `CampaignRun` Django model — the persistent record that replaces the ad-hoc 3I/ATLAS
Google Sheet — carrying the full sheet field inventory, a required campaign `TargetList` FK, an
optional observed `Target` FK, and a two-field controlled-vocabulary status (`approval_status` +
`run_status`). Generate its migration and prove the schema with model-level tests.

Purpose: This is the data-layer foundation for the entire v2.0 milestone. Phases 15-17 (table view,
submission form, coverage-gap) all read/write this model. Getting the field inventory and status
vocabulary right now — validated against the real sheet's shape — is the whole point of ordering the
model-and-import phase first.
Output: `CampaignRun` model + `ApprovalStatus`/`RunStatus` TextChoices, migration `0002_campaignrun.py`,
and `test_campaign_models.py`.
</objective>
