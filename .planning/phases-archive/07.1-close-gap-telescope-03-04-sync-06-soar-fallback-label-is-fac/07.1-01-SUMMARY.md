## Decisions Made

- **Final signature:** `_coarse_telescope_label(instrument_type: str, facility_name: str) -> str`.
- **SOAR-detection predicate:** `facility_name.upper() == 'SOAR'` — an exact, case-insensitive match against `record.facility`. No substring or prefix matching was used, since `record.facility` is a fixed enum-like string (`'LCO'`/`'SOAR'`) set by TOM Toolkit's own facility dispatch, not free-form input.
- **Call-site argument:** `record.facility` (the string), explicitly not the `facility` parameter of `_build_event_fields` (an `LCOFacility` instance) — this was the single highest-risk detail flagged in the plan's `<action>` section and was verified by grep (`_coarse_telescope_label(.*record.facility` matches) plus the full passing test suite.
- Docstring updated to cite TELESCOPE-03, the v1.3 milestone audit, and the single `('sor','4m0')` `SITE_TELESCOPE_MAP` entry as the source of the "SOAR is unconditionally 4m0" fact, per the project's formula/decision-citation convention.
