<objective>
Build the `load_telescope_runs` Django management command and its test suite.
The command reads a text file of classical-schedule run lines, parses each via
Phase 2's `parse_run_line()`, expands each `ParsedRun` into one
`tom_calendar.CalendarEvent` per observing night using Phase 1's
`get_site()`/`sun_event()`, and upserts those events idempotently keyed on
`(telescope, instrument, start_time)`. Per-line parse/lookup errors are logged
to stderr and skipped without aborting the run. No new models or migrations.

Purpose: Delivers all three phase requirements (INGEST-01/02/03) â€” the entire
calendar-ingest behavior lives in this one command file.
Output: `solsys_code/management/commands/load_telescope_runs.py` and
`solsys_code/tests/test_load_telescope_runs.py`.
</objective>

Signatures this plan consumes (already implemented, do not modify):

- `parse_run_line(line: str) -> ParsedRun` â€” raises `ValueError` on empty
  line, ambiguous/unknown telescope (D-01), unrecognized status, or no date
  range found.
- `ParsedRun` (frozen dataclass) fields: `telescope: str` (resolved SITES key),
  `instrument: str`, `status: str` (lowercase, default `'allocation'`),
  `year: int`, `month: int`, `day1: int`, `day2: int`.
- `get_site(name: str) -> Observatory` â€” `name` is a `ParsedRun.telescope`
  value; raises `Observatory.DoesNotExist` if no row exists for the obscode.
- `sun_event(site: Observatory, date: datetime.date, kind: str) -> tuple[Time, Time]`
  â€” `kind='sun'` returns `(sunset, sunrise)` dip-corrected; `kind='dark'`
  returns `(dark_start, dark_end)` at -15 deg. Returns `(setting, rising)` in
  chronological order. Raises `ValueError` if the sun does not cross the
  threshold exactly twice (high-latitude / no-darkness nights).
- `astropy.time.Time.to_datetime(timezone=datetime.timezone.utc) -> datetime`
  â€” converts the returned `Time` objects to timezone-aware UTC datetimes.

`telescope = CharField(max_length=200, blank=True, default="")`,
`instrument = CharField(max_length=200, blank=True, default="")`,
`modified = DateTimeField(auto_now=True)`. No unique constraint beyond `pk` â€”
